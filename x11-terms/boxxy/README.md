# boxxy

This package is currently on hold.

## Status

**Blocked**: Waiting for `>=gui-libs/gtk-4.21` (stable 4.22) and `>=gui-libs/libadwaita-1.9`.

The current upstream version of `boxxy` requires features introduced in these versions of GTK and Libadwaita. As of April 2026, the main Gentoo tree only provides up to GTK 4.20.3 and Libadwaita 1.8.2.

Compilation tests (`ebuild compile`) will fail until the system dependencies are updated.
